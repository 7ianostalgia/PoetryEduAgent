## [0.2.0] - 2026-06-22

### 新增

- 教师端与学生端角色化 Web 工作区。
- 多 Agent 实时协同大盘、增量事件接口和 SSE。
- SQLite 运行仓库，持久化任务、事件、资源、审核、反馈和学习报告。
- 历史任务查询与结果恢复。
- 教师指定模块反馈与定向资源修订。
- 教师资源包下载。
- 学生四题测评、Qwen rubric 评分与个性化学习报告。
- DeepSeek-V4-Flash 文字审核、本地 Qwen fallback 和单轮文字纠偏。
- Qwen-VL 短文本图片观察、确定性图片门禁和单轮 Prompt 纠偏重绘。
- Kolors 结构化 Prompt 编译与生成参数记录。

### 改进

- 文字审核与图片审核采用独立输入，避免两个质量维度互相影响。
- 最终结果明确区分任务完成、文字门禁、图片门禁和双门禁通过。
- 前端展示 gpu 阶段和可读错误，不将模型失败伪装成成功结果。
- 教师资源包不暴露生图 Prompt 和原始技术审核信息。
- 教师端与学生端生成图片固定按 `1:1` 比例展示。

## [0.1.0] - 2026-06-20

### 新增

- FastAPI 学习任务、结果和测评接口。
- 可版本化 SQLite 诗词知识库及旧库只读迁移工具。
- 标题、诗句和知识内容 RAG 检索。
- Qwen2.5-14B-Instruct-AWQ 结构化文本客户端。
- Kolors 单图生成客户端。
- Qwen2.5-VL 图片审核客户端。
- 单 GPU 模型调度、请求限制和独立子进程基础设施。
- JSON Schema 校验、语义护栏和完整自动化测试基线。
